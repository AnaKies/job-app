# Индекс архитектуры

Этот каталог содержит утвержденную архитектуру программного обеспечения для CeeVee.

## Документы

- [overview.md](./overview.md)
  Объясняет направление архитектуры, управляющие принципы, выбранную форму runtime и основные компромиссы.

- [system-context.md](./system-context.md)
  Описывает основных акторов, границы системы и поток взаимодействия end-to-end.

- [module-design.md](./module-design.md)
  Определяет внутреннюю структуру модулей, шестиугольные границы и владение основными компонентами.

- [interfaces.md](./interfaces.md)
  Документирует основные внутренние и внешние интерфейсы, включая инструменты MCP и контракты адаптеров.

- [data-model.md](./data-model.md)
  Определяет основные сущности, взаимосвязи, жизненный цикл данных и обязанности векторного поиска.

- [runtime-observability.md](./runtime-observability.md)
  Описывает поведение runtime, фоновую обработку, требования к надежности и операционную видимость.

## Взаимосвязь между документами

- `overview.md` является точкой входа для архитектурных решений.
- `system-context.md` объясняет, как внешние акторы взаимодействуют с системой.
- `module-design.md` объясняет, как система структурирована внутри.
- `interfaces.md` определяет контракты, которые соединяют модули и внешних потребителей.
- `data-model.md` определяет постоянную модель, используемую модулями runtime.
- `runtime-observability.md` объясняет, как архитектура ведет себя в операции.

Все документы в этом каталоге описывают только текущее утвержденное состояние архитектуры.
